CSS Pseudo Class

What is a Pseudo Class?

A pseudo-class is a keyword added to a selector that specifies a special state of the selected elements.

For example, a link can be styled differently when a user mouses over it or when it has been visited.

Examples of Pseudo Classes

  • :hover - when the user mouses over an element
  • :active - when the user clicks on an element
  • :focus - when an element has focus (like an input field)
  • :visited - when a link has been visited
  • :first-child - selects the first child of a parent element
  • :last-child - selects the last child of a parent element
